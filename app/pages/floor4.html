Adoben ohjelmat


Tieto- ja viestintätekniikan opinnoissa ja erityisesti Mediatekniikalla, tulevat monet Adoben ohjelmat tutuiksi.


Suurimmasta osasta Dynamon LabranNetin koneista löytyy Adoben ohjelmat. Jos haluat ladata ohjelmat kotikoneellesi opiskelijoille on erikoistarjous Adobelta vain 20,15 €/kk sis. ALV !

Tässä on Tekninen matematiikka 1 -kirjasta lainattu tehtävä. Kurssi tulee ensimmäisen vuoden keväänä monille tieto- ja viestintätekniikan opiskelijoille. Osaisitko sinä ratkaista tehtävän?


Syötä tähän oikea vastaus:



using System;
public class Hello
{
public static void Main()
{
Console.WriteLine("Hello, World!");
}
}

Ohjelmointi

Tieto- ja viestintätekniikan ensimmäisen vuoden opiskelijoista kaikki tulevat kokeilemaan ohjelmointia ensimmäisen vuoden syksystä. Sen jälkeen toisilla tulee olemaan enemmän, ja toisilla vähemmän ohjelmointia, riippuen suuntautumisesta. Ensimmäisen vuoden aikana monet tulevat tutustumaan C++ ja C#.


Vieressä on tyypillinen ohjelmoinnin ensimmäinen esimerkki Hello World, kieli on C#.
